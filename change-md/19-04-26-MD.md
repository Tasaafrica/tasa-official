## Fixed Issues

### Middleware Redirect to Dashboard (404 Error)

**Problem:** Absolute URL redirects in Next.js middleware were causing 404 errors when trying to redirect `/profile` to `https://dash.tasa.com.ng/user/{id}/profile`

**Solution:**

- Created `src/app/profile/route.ts` - Route handler that properly handles server-side redirects
- Removed problematic middleware redirect code from `src/middleware.ts`
- Uses `getServerSession()` to access session data reliably
- Redirects authenticated users to dashboard, unauthenticated users to signin

**Files Changed:**

- ✅ Created: `src/app/profile/route.ts`
- ✅ Modified: `src/middleware.ts` (removed lines 26-30)
